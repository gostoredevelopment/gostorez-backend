# MVP Feature List

## MUST HAVE
- Vendor profile creation & basic verification  
- Product listing (images, price, description)  
- Ordering (cart + checkout)  
- Payments (Paystack / Flutterwave integration)  
- Authentication (email / phone)  
- Storage for product images  
- Basic chat (text + images)  
- Vendor dashboard (view orders & products)

## SHOULD HAVE
- Vendor wallet and balance  
- Product search + category filter  
- Push notifications (FCM)  
- Order tracking updates  
- NIN verification (stub)

## CAN HAVE (later)
- Video calls (Daily.co)  
- 24-hr status updates  
- Hot Ads automation  
- Referral & loyalty programs
